Maya Script Store

This website contains a collection of scripts for Autodesk Maya that I have written or adapted. You can access them by completing surveys which should only take a few minutes each; it's much friendlier than a regular paywall while paying out a lot more than banner ads.

Register an account with the 'Register' button above and then log in with the provided user id. Afterwards, click on the Pollfish widget in the bottom right to be served a survey; successfully completing the survey will reward you with between 30 and 100 points that you can use to unlock content.
